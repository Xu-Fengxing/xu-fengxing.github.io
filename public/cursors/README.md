# Hachiware光标主题使用指南

## 您的光标主题信息
- **主题名称**：Hachiware
- **源文件位置**：`D:\Pictures\Cursors`
- **文件数量**：15个.ani文件
- **目标文件夹**：`public\cursors`

## 转换步骤

### 方法1：在线转换（推荐）
1. 访问 https://convertio.co/ani-png/
2. 批量上传您的.ani文件
3. 选择PNG格式
4. 下载转换后的文件

### 方法2：使用专业工具
- **Cursor Icon Editor** - 专业光标编辑工具
- **Axialis CursorWorkshop** - 商业级工具
- **RealWorld Cursor Editor** - 免费工具

## 文件放置

将转换后的PNG文件**保持原文件名**放入 `public\cursors` 文件夹：

```
public/cursors/
├── 1 (1).png      # 默认光标
├── 1 (15).png     # 帮助光标
├── 1 (16).png     # 对角调整1
├── 1 (17).png     # 对角调整2
├── 1 (18).png     # 工作光标
├── 1 (19).png     # 移动光标
├── 1 (20).png     # 垂直调整
├── 1 (21).png     # 禁用光标
├── 1 (22).png     # 文本光标
├── 1 (23).png     # 水平调整
├── 1 (24).png     # 十字光标
├── 1 (25).png     # 等待光标
├── 1 (26).png     # 向上箭头
├── 1 (27).png     # 链接光标
└── 1 (29).png     # 手写光标
```

## 注意事项
1. **保持原文件名**：不要重命名，CSS中已配置好对应关系
2. **PNG格式**：建议32x32像素，支持透明背景
3. **文件完整性**：至少需要 `1 (1).png` 和 `1 (27).png` 两个核心文件
4. **浏览器兼容**：现代浏览器都支持PNG光标
